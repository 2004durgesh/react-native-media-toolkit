# Hooks

### `useVideo()`

A hook to access the current video player state and dispatch actions to control the player.

```tsx
import { useVideo } from 'react-native-media-toolkit';

const MyComponent = () => {
  const { state, actions, setTheme } = useVideo();
  // ... use state and actions
};
```

### `useTheme()`

A hook to access the currently active theme.

```tsx
import { useTheme } from 'react-native-media-toolkit';

const MyComponent = () => {
  const { colors } = useTheme();
  // ... use theme colors
};
```

### Other Hooks

- `useBuffering()`: Checks if the video is currently buffering.
- `useControlsVisibility()`: Manages the visibility of player controls.
- `useFullscreen()`: Manages fullscreen state.
- `usePlayback()`: Provides playback state (playing, paused, ended).
- `useProgress()`: Provides current playback progress and duration.
- `useVolume()`: Manages volume state.

For more details on these hooks and their return values, please check the `src/hooks` directory.
