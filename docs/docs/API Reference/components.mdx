# Components

This document provides an overview of the core components available in `react-native-video-toolkit`.

## `VideoPlayer`

The `VideoPlayer` is the main component that wraps the video surface and provides a context for all other player controls and UI elements.

```tsx
<VideoPlayer source={videoSource} containerStyle={{}} videoProps={{}} gestureProps={{}}>
  {/* Children components, e.g., layouts or custom UI */}
</VideoPlayer>
```

| Property         | Type                                                          | Description                                                                |
| ---------------- | ------------------------------------------------------------- | -------------------------------------------------------------------------- |
| `source`         | [`VideoSource`](./Types/video#videosource)                    | The source of the video to be played.                                      |
| `containerStyle` | `StyleProp<ViewStyle>`                                        | Custom styles for the player container.                                    |
| `videoProps`     | `ReactVideoProps`                                             | Additional props to pass to the underlying `react-native-video` component. |
| `gestureProps`   | [`GestureHandlerProps`](./Types/gestures#gesturehandlerprops) | Props to configure gesture handling (e.g., double-tap, pan gestures).      |

## `VideoPlayer.Controls`

A compound component used to wrap and position your custom control elements within the `VideoPlayer`.

```tsx
<VideoPlayer.Controls style={{}}>{/* Your control components */}</VideoPlayer.Controls>
```

## Basic Controls

The library provides a set of ready-to-use basic control components that can be used directly within `VideoPlayer.Controls` or any custom layout.

- **`VideoPlayer.PlayButton`**: A button to play or pause the video.
- **`VideoPlayer.ProgressBar`**: A slider for seeking through the video.
- **`VideoPlayer.TimeDisplay`**: A component to display the current time and duration of the video.
- **`VideoPlayer.VolumeControl`**: A slider for adjusting the video volume.
- **`VideoPlayer.FullscreenButton`**: A button to enter or exit fullscreen mode.
- **`VideoPlayer.MuteButton`**: A button to mute or unmute the video.
- **`VideoPlayer.LoadingSpinner`**: A spinner to indicate that the video is loading.
