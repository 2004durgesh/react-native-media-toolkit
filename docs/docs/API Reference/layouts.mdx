# Layouts

Pre-built layouts to quickly set up a video player UI.

### `DefaultLayout`

A comprehensive layout with standard controls.

```tsx
<VideoPlayer source={videoSource}>
  <DefaultLayout />
</VideoPlayer>
```

### `MinimalLayout`

A minimalist layout with essential controls.

```tsx
<VideoPlayer source={videoSource}>
  <MinimalLayout />
</VideoPlayer>
```
